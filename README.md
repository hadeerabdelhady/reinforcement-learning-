# Reinforcement Learning

Reinforcement Learning is a branch of Machine Learning that focuses on how agents learn to make decisions through interaction with an environment. The objective is to learn actions that maximize cumulative rewards over time by balancing exploration and exploitation.

In this course, fundamental concepts of sequential decision-making were introduced, including the **Multi-Armed Bandit Problem** and **Markov Decision Processes (MDP)**. The course also covered the **Bellman Equation**, which provides the mathematical foundation for evaluating and improving decision policies.

Several approaches for learning optimal policies were explored, including **Dynamic Programming**, **Temporal Difference Learning**, **SARSA** as an on-policy learning method, and **Q-Learning** as an off-policy learning approach.

This repository contains assignments and implementations that demonstrate these reinforcement learning concepts and algorithms through practical applications.
